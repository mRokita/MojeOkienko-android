# Moje Okienko - Wersja dla androida
## Projekt na BIHAPI 2015
### Testowanie
Aplikację można zainstalować z dwóch źródeł;     
* Ręcznie skompilowany kod źródłowy
    1. Pobranie kodu
        * Sklonowanie kodu z repozytorium GIT
        
        ```
        git clone git@git.mrokita.pl:MojeOkienko/android.git
        ```
        * Pobranie archiwum ZIP z GitLab'a (https://git.mrokita.pl/MojeOkienko/android/repository/archive.zip)
    2. Kompilacja z użyciem IDE wspierającego development dla systemu Android (np. Android Studio)
* Pobranie [archiwum APK](app/build/outputs/apk/app-debug.apk)
